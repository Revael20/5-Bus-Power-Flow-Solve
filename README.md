The solver is built using the Gauss-Seidel numerical method. Key technical steps include:
Admittance Matrix ($Y_{bus}$): Full construction including series resistance, reactance, and shunt susceptance (charging capacitance) for accurate loss calculation.
Vectorized Computation: Optimized MATLAB loops to handle $N$ buses and $M$ generators.
Convergence Analysis: Iterative solving with a strict tolerance of $1 \times 10^{-9}$ to ensure high precision.
Multi-Platform Verification: Every result generated by the MATLAB script is benchmarked against PowerWorld Simulator to verify the model's accuracy in calculating nodal voltages and system-wide power flows.
