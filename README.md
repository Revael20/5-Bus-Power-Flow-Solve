ðŸ§ª The Core Challenge: PV-PQ Switching
In power systems, generators (PV buses) have physical limits on how much reactive power ($Q$) they can provide. If the grid demand exceeds these limits, the generator can no longer maintain its voltage setpoint55. This project implements Dynamic Logic that:
Detects $Q_{max}/Q_{min}$ limit violations during iterations.
Automatically converts the bus to a PQ-Limited type to maintain physical feasibility.
Recovers to PV mode if the system state returns within operating bounds8.

ðŸ›  Technical Implementation
The solver compares two primary numerical methods:

1. Gauss-Seidel (GS) Solver:

Uses sequential updates based on nodal current equations.
Converged in 357 iterations for this 5-bus system.
Lower memory overhead but slower linear convergence.

2. Newton-Raphson (NR) Solver:
Uses a linearized Jacobian matrix ($J_1$ through $J_4$) and Taylor series expansion.
Incorporates the "Q-substitution trick" for efficient Jacobian diagonal derivation ($H_{ii}$ and $L_{ii}$)131313.
Converged in only 7 iterations, demonstrating superior quadratic convergence1414.
